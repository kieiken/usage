## Modified Version

This version was adapted as a secondary modification for Claude Desktop usage display.
The main change is that the app can prefer `~/.claude/usage-status-derived.json`, a local status file derived from Claude Desktop logs, instead of relying only on the Claude Code CLI statusLine hook.
